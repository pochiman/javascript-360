## Objective

    Write a function that takes in a string and returns the longest word in the array.

## Example Returns

- yourFunction('Ready, steady, go!') will return **'steady'**.
- yourFunction('Ready[[[, steady, go!') will return **'steady'**.
- yourFunction('ABCd') will return **'ABCd'**.
